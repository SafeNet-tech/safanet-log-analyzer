# 🛠️ SafeNet Log Analyzer (v1.0.0)
> Utilitário autónomo em Python para automação de auditorias e eficiência operacional em infraestruturas de CCTV e Redes.

## 📌 O Problema Operacional
Instaladores, ISPs e administradores de sistemas perdem diariamente entre 30 a 45 minutos a analisar manualmente ficheiros brutos de logs (`.txt` / `.log`). Identificar quebras de sinal, tentativas de invasão ou timeouts de hardware no meio de milhares de linhas de dados normais atrasa o tempo de resposta técnica.

## 💡 A Solução SafeNet
O **SafeNet Log Analyzer** elimina o trabalho manual. O script faz o varrimento de ficheiros pesados em milissegundos, elimina dados redundantes de sucesso e isola estritamente os eventos críticos que exigem a atenção do técnico.

## 📊 Benefícios para a Operação da Sua Empresa
* **Alta Velocidade:** Processamento instantâneo (+2.000 linhas por segundo).
* **Precisão Técnica:** Zero falsos positivos; isolamento direto por Timestamp (Hora), IP e tipo de falha.
* **Relatório Limpo:** Gera automaticamente um ficheiro formatado (`relatorio_erros.txt`) pronto para ser enviado à gerência ou arquivado como histórico de auditoria.
* **Sem Dependências:** Corre nativamente em qualquer máquina com Python instalado, sem necessidade de internet ou pacotes externos.
